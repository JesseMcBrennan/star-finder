### Endpoints

* ##### GET - Stars `/api/v1/stars`

	Using this endpoint will return you an array containing all star objects - ex: [{id: 1, name: '11 Com', mass: '123.1231'}, {id: 2, name:'ASDFASD', mass: '123.11'},...]
    
* ##### GET - Stars `/api/v1/stars/:id`
